# Projeto 1 : Planejamento de um aplicativo para abrigo de animais.

## 📒 Descrição
O aplicativo prentende atender as principais funcionalidades de um abrigo de animais de rua.

## 🤖 Tecnologias Utilizadas
IA Generativa usada :  [GAMMA](https://gamma.app/)  

## 🧐 Processo de Criação
O prompt utilizado foi :

- Planeje um aplicativo para uma organização de abrigo de animais de rua.
Precisa ter opções para administrar o cuidado medico dos pets, assim como a criação de eventos de adoção, controle de estoque de produtos e medicamentos, cadastro de funcionários, médicos veterinários, voluntários; e outras funcionalidades pertinentes.
Use conceitos de gamification para a melhor experiência de usuário.
Inclua em cada cartão uma imagem de gato ou cachorro. 

## 🚀 Resultados
O resutado foi o e-book "Introducao-ao-aplicativo-de-abrigo-de-animais.PDF" que está neste reposritório.
